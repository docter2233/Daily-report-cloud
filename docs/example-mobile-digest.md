## 论文

1. Efficient Emotion-Aware Iconic Gesture Prediction for Robot Co-Speech
- 主旨: 这篇论文关注机器人共语手势生成，目标是让机器人表达更自然。
- 方法: 主要采用 Transformer 路线做序列预测。
- 理论: 理论抓手偏向生成式时序建模，而不是传统控制器设计。
- 实验: 在公开数据集上评估。

2. Diffusion-Based mmWave Radar Point Cloud Enhancement Driven by Range Images
- 主旨: 这篇论文关注雷达感知与点云增强。
- 方法: 采用扩散模型和几何/深度估计结合的路线。
- 理论: 更像感知表示建模，而不是显式动力学推导。
- 实验: 在公开基准上评估。
