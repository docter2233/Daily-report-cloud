# Code of Conduct

This project aims to be useful for researchers, engineers, and contributors working on robotics and embodied AI.

## Expected behavior

- Be respectful and constructive
- Critique ideas, not people
- Keep discussions technical and actionable
- When reporting issues, provide enough context to reproduce or understand the problem

## Unacceptable behavior

- Harassment or personal attacks
- Hostile or dismissive behavior
- Spam, trolling, or repeated low-signal issue filing
- Encouraging piracy-based workflows in this repository

## Scope

This applies to issues, pull requests, discussions, and review comments in this repository.

## Maintainer discretion

Project maintainers may remove comments, close issues, or block contributors whose behavior repeatedly harms collaboration quality.
